寒いですね！みなさま新年いかがお過ごしでしょうか。  
ハウスにも若干の正月らしさを加えるため、スーパーで買った簡易な鏡餅セットを飾ってみました（笑）。金粉入りのお酒も一緒に。

実は１月１日でヘッドクォーターズ神戸の運営が切り替わってちょうど１年が経ったことになります。  
あっという間ですね。

前オーナーによりゲストハウス半分＋シェアハウス半分のような運営がされていたゲストハウス時代は外国からの宿泊客で賑わったり素敵な感じでしたが、今は長期の住人たちでとても仲良く暮らしています。

近くの神戸大学・大学院の元／現の学生が今は３人（ちょっと前までは４人）も住んでくれてたり、内見に来てくれた方には写真でハウス内の雰囲気を伝えるととても興味をもってくれて住みたいと言ってくれることが増えたり、すこしずつ良い循環がまわるようになってきたかなぁと、オーナーとしては嬉しく思っています。

おかげさまでほぼ満室が続いています。（あとひとり、できれば男性が来てくれるとバランスいいなぁ）

ハード面としては以前の状態から部屋を２つ増やしたり、家具家電を増やしたり、屋上を綺麗にしたりなどの整備も多少はしましたが、大事なことは住んでいる住人の良い人間関係や「ここに住んで良かった」と思える環境をつくるソフト面ですね。

今年もまたさらに素敵なハウスにできるよう、イベント的なものを増やすなど内部・外部の交流がより活発になるようなことが出来たらいいなと思っています。

それでは２０２０年を良い年にしていきましょう😊  
よければみなさん遊びに来てくださいね。
