料理の準備を１５時から始めて、皆で手分けして水道筋商店街で買い物や調理。ハンガリーカップルの彼氏ピーターが料理男子なのでハンガリーとイギリスの家庭料理を提案してくれていっぱい作りましたよ（複数メニューのレシピをPCで書き出して材料の購入量を真剣に決めてた）。

*   ![](https://theheadquarters.jp/wp-content/uploads/2019/12/RyouriBoys-1024x682.jpg)
    
*   ![](https://theheadquarters.jp/wp-content/uploads/2019/12/DSCF2795-1024x683.jpg)
    

かぼちゃのスープはシンプルなのに高級料理店のような味で美味しかったです😆。そして日本料理を代表する一品としてたこ焼きも（笑）。あんまりうまく焼けなかったけどまあいいか。

そして年末で残念ながら退去になってしまうJ君の送別会も兼ねてたので、クイズで宝探しをさせてプレゼント！（あげた写真は当日のものをリアルタイムでコンビニでネット印刷）。喜んでもらってよかった。  
パーティのあとは、お風呂の順番を決めるための恒例あみだくじもね。
