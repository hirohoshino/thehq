いまヘッドクォーターズには一番若い人で16歳高1男子が、多くは20代の男女が、そして外国人ではハンガリーのカップルも暮らしています。

住人が普段楽しそうに暮らしていたとしても、それが親御さんとかに伝わっているのかな、シェアハウスの暮らしぶりに不安もたれてないかなとふと心配になりました。  
みんなに聞いてみたところ、けっこうみんなハウス内の写真を親に送っていたり、また意外と家族を呼んでハウス内を案内したことがあるそうで、安心しました😊。屋上を見せたらおばあちゃんがすごく喜んでたとかもｗ。

そして海を渡ってきたハンガリアンズは、実はYouTubeで楽しいクリスマスの動画メッセージを作っていたって。許可をもらって転載。みんなでハンガリー語の歌を歌わされたのはこのためだったんですねｗ。いいですねこういうの！
